Table Management Application (Ford Motor Credit Company)

Every application has a set of table associated with it and TMA application is used to maintain and administer those tables. It uses the TMA framework developed by ATD. The administration application is a fully functioning application that maintains table profiles. A table profile defines titles for columns, which columns should be displayed, and the sequence that columns should be displayed in when maintaining table data. In addition, the profile specifies the class used to validate data and the class used for database access. The security functionality runs "out of the box". Each table is assigned function codes for data access. If a user has the appropriate function code in his/her SDA, then the user has access to that table. The maintenance portion of the application handles all of the steps necessary to maintain data in a table. The application provides for data entry, data validation and a save review prior to updating the database.
